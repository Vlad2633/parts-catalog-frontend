```md
# Parts Catalog — Frontend

Клієнтська частина курсового проєкту **«Каталог автозапчастин»**.

Веб-додаток для перегляду автозапчастин, роботи з корзиною та оформлення замовлень.

---

## Технології
- React
- Vite
- React Router
- Axios
- CSS (custom UI)

---

## Функціонал
- Каталог автозапчастин
- Пошук та фільтрація
- Перегляд детальної інформації
- Корзина (localStorage)
- Оформлення замовлення
- Авторизація та реєстрація
- Адмін-панель (для ролі ADMIN)

---
```
